Minecraft plugin development utilities.

## Installation

### Maven

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

```xml
<dependency>
    <groupId>com.github.putindeer</groupId>
    <artifactId>mcdev-utils</artifactId>
    <version>1.0.10</version>
</dependency>
```

### Gradle

```gradle
repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.putindeer:mcdev-utils:1.0.10'
}
```
